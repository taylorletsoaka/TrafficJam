#
App
==============

Traffic Jam is a car based game where the goal is get a red car out of the traffic by driving it to the right end of the board

Dependencies
========

```stdlib.jar```

Workflow
========

To compile the entire project,

```javac RushHour.java```


```java RushHour config```
